Visual Studio Code

Visual Studio Code (VS Code) is a free, streamlined code editor that helps developers write and work on code:

Features

Supports a variety of programming languages, including JavaScript, TypeScript, and Node.js. It also includes features like debugging, syntax highlighting, code completion, and version control.

Ease of Use

VS Code is easy to install and use, and it doesn't take up too much space. It's a good choice for beginners, and it's also useful for advanced programmers.

Customization

You can change the theme, keyboard shortcuts, and preferences. You can also add extensions to do more, like work with different programming languages or tools.

Platform Support

VS Code works on Windows, Linux, and macOS.

Community

VS Code is based on the MIT licensed program "Visual Studio Code — Open Source" (also known as "Code — OSS").

Editions

VS Code has three editions: community, professional, and enterprise. The community version is free, while the professional and enterprise are not.
Here are some other things you can do with VS Code:

  • Run tasks
  • Use snippets
  • Refactor code
  • Use embedded version control with Git
  • Select multiple files and run actions on them
  • Use the Compare Selected command to diff two files

Types of Emmets

Emmet Symbol Description
Child > You can use the `>` operator to nest elements inside each other.
Ex: div>ul>li
Sibling + Use the `+` operator to place elements near each other.
Ex: div+p+bq
Climb-up ^ With the `>` operator, you’re descending down the generated tree and resolving positions against the deepest element.
Ex: div+div>p>span+em^bq
Multiplication * Use the `*` operator to repeat elements.
Ex: ul>li*5
Grouping () Parentheses are used by Emmet power users for grouping subtrees in complex abbreviations.
Ex: div>(header>ul>li*2>a)+footer>p
Item Numbering $ With the multiplication `*` operator, you can repeat elements, and with `$`, you can number them.
Ex: ul>li.item$*5
Text {} You can use curly braces to add text to an element.
Ex: a{Click me}
